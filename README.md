# Axios API Authentication

This project demonstrates how to use Axios in a Node.js Express application to interact with an external API using different authentication methods.

## Features

- No Authentication request
- Basic Authentication
- API Key Authentication
- Bearer Token Authentication

## Technologies Used

- Node.js
- Express.js
- Axios
- EJS

## Endpoints

- `/noAuth` – Fetches a random secret without authentication.
- `/basicAuth` – Retrieves secrets using Basic Authentication.
- `/apiKey` – Filters secrets using an API key.
- `/bearerToken` – Retrieves a specific secret using a Bearer token.

## Installation

```

npm install

```

## Run the Project

```

node index.js

```

Then open:

```

[http://localhost:3000](http://localhost:3000)

```

## Purpose

This project was built as part of a Full Stack Development course to practice making API requests and implementing different authentication methods using Axios.
